### cd [디렉토리명]
- Change Directory
- 디렉토리를 이동할 때 사용

#### 옵션  
cd [엔터], cd ~, cd ~/, cd $HOME : 사용자의 홈 디렉토리로 이동  
cd . : 현재 디렉토리로 이동  
cd / : 최상위 디렉토리로 이동  
cd $변수명 : 변수에 지정된 경로로 이동  
cd ~계정명 : 입력한 사용자의 홈디렉토리로 이동  
cd - : 이전 경로로 이동
